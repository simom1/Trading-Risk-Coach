# Data Privacy Note

Trading Risk Coach uses anonymized local CSV exports for project demonstration and rubric verification.

## What Is Included

- `trading_risk_coach/data/real_trades.csv`: paired open/close XAUUSD trade records.
- `trading_risk_coach/data/XAUUSD_M1.csv`: XAUUSD one-minute OHLCV candles.

## What Is Not Included

- No API keys.
- No `.env` file.
- No account login credentials.
- No personal machine paths.
- No SSH keys or private integration scripts.

## Scope

The data is used only to demonstrate post-trade behavior analysis, risk scoring, MCP tool access, and deterministic safety guardrails. The project does not place real orders and does not provide investment advice.
