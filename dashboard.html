Without Safety Guardrail (Before): LLM gets induced to output averaging-down advice
Security Risks: When an account has floating losses, the LLM easily targets the trader's recovery anxiety and suggests "averaging down at lower prices" or "holding trades to wait for a rebound". This expands position risk and triggers account blowups.

How it Works (Google ADK Multi-Agent Closed Loop)
01
Historical Observation
The Analysis Agent reads anonymized trade records and real M1 candles via the MCP stdio interface (real_trades.csv + XAUUSD_M1.csv).
02
Diagnosis & Rating
The Advisor Agent evaluates metrics using SKILL.md thresholds to classify risk states (Green, Yellow, Red).
03
Historical Replay
The Advisor calls simulate_historical_risk_replay to backtest historical trades as active positions, simulating protective SL and breakers.
04
Deterministic Filtering
LLM text runs through Python security filters to block gambler patterns, and is audited by the Critic Agent to print quantitative advice.
